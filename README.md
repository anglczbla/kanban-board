A modern and intuitive Kanban board application for managing your tasks and projects efficiently.

## Features

- **User Authentication** - Secure login and registration system
- **Drag & Drop** - Easily move tasks between columns with smooth animations
- **Task Management** - Create, edit, and organize tasks across different stages
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Real-time Updates** - Stay synced with the latest task changes

## Tech Stack

- **Frontend Framework:** React 19
- **Routing:** React Router DOM
- **State Management:** TanStack React Query
- **Drag & Drop:** dnd-kit
- **Styling:** Tailwind CSS 4
- **HTTP Client:** Axios
- **Build Tool:** Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

1. Register a new account or login with existing credentials
2. Create new tasks by clicking the add button
3. Drag and drop tasks between columns (To Do, In Progress, Done)
4. Edit or delete tasks as needed
5. Track your progress across different project stages
